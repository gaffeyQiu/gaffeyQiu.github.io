VsCode 使用技巧

Nov 17, 2021
1 minute read

常用

  • cmd + p 打开命名面板(万能)
  • ctrl + tab 切换标签页
  • ctrl + shift + tab 反向切换标签页
  • ctrl + 丶 打开显示终端

代码相关

  • alt + F8 光标定位到当前页面问题代码
  • cmd + . 快速修复问题代码
  • ctrl + Enter 水平分屏方式打开文件
  • Go 语言静态检查配置
    • staticcheck是针对go的静态检查工具，文档地址
    • 在 setting.json 中过滤掉对应的编号(例如过滤掉 ST1020 和 所有 S1 开头的规则) 
      "go.lintFlags": [
		"-checks",
		"all,-ST1020,-S1*"
	]

vim 相关

  • ds" 删除 “abc” 的双引号
  • cs"' 替换 “abc” 为 ‘abc’
  • ysw" 添加"“给一个单词
  • S" 选中 abc 的情况下, 给 abc 加上双引号
