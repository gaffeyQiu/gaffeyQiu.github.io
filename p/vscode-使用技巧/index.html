VsCode 使用技巧

Nov 17, 2021
1 minute read

vscode 配置或快捷键

  • cmd + j 打开或关闭终端
  • F12 或 ctrl + ] 跳转到函数定义处
  • gh 显示鼠标 hover 在当前位置时提示的信息
  • ctrl + i 前进
  • ctrl + o 后退
  • alt + F8 光标定位到当前页面问题代码
  • cmd + . 快速修复问题代码
  • ctrl + Enter 水平分屏方式打开文件

vscode json 配置项

  • Go 语言静态检查配置
    • staticcheck是针对go的静态检查工具，文档地址
    • 在 setting.json 中过滤掉对应的编号(例如过滤掉 ST1020 和 所有 S1 开头的规则) 
      "go.lintFlags": [
		"-checks",
		"all,-ST1020,-S1*"
	]

vim 自身按键

  • {number} + w 向前移动 N 个单词
  • {number} + b 向后移动 N 个单词
  • % 跳转到匹配的括号
  • ctrl + ] + gd 跳转到函数定义处
  • gh 显示鼠标 hover 在当前位置时提示的信息
  • {number} + f + {char} 向右移动到第 N 个指定字符处
  • zt/zz/zb 光标跳转到屏幕的顶/中/底部
  • {number} + ctrl + E 向上滚屏
  • {number} + ctrl + Y 向上滚屏
  • ctrl + c 退出插入模式

vim 插件快捷键

  • ds" 删除 “abc” 的双引号
  • cs"' 替换 “abc” 为 ‘abc’
  • ysw" 添加"“给一个单词
  • S" 选中 abc 的情况下, 给 abc 加上双引号
